# 基于workerman的聊天室

##### 首先  启动php内置服务器 默认端口为9999

~~~php 
 php -S localhost:9999
~~~

##### 然后启动websocket服务

~~~php
php server.php start
~~~

##### 在dologin.php 中修改数据库的名称、用户名、密码

##### 转储chat_user.sql文件

##### 登陆 账号1  jack 密码 123   账号2  tom 密码 123

##### 点击右边用户列表 进入私聊
